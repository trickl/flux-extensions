Utility library with flux publishers, subscribers and transformers

Installation
============

To install from Maven Central:

```xml
<dependency>
	<groupId>com.github.trickl</groupId>
	<artifactId>flux-extensions</artifactId>
	<version>0.0.1-SNAPSHOT</version>
</dependency>
```
